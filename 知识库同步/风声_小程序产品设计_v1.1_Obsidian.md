## 核心定位（v1.1更新）

**小程序 = 网页版的"现场版"**

- 网页版 = 展示 + 行业背书 + 机构合作（面向行业/高校/B端）
- 小程序版 = 随身工具 + 高频使用 + 个人赋能（面向一线经纪人/服务者）
- 两者关系：互补而非替代，数据完全打通

## 功能架构（v1.1更新）

### 首页：4个入口（砍个人中心）

- 六维品质测评（最大卡片，获客入口）
- 客户解码器 + 带看报告（并排）
- 客户跟进记录（小程序独家，突出展示）
- 知识底座（底部入口）

### 5个核心功能

1. **六维品质测评**：精简到10题1分钟 + 海报分享 + 微信分享
2. **客户解码器**：最近使用置顶 + 搜索 + 语音输入 + 收藏 + 分享
3. **客户跟进记录**（小程序独家·杀手级应用）：写跟进 + 设提醒 + 客户档案 + 标签分类
4. **带看报告**：拍照+语音录入 + PDF导出 + 房源库 + 带看日历
5. **AI助手**（小程序独有）：语音对话 + 场景推荐 + 每日一学 + 随身顾问

### 客户跟进记录与现有工具联动

- 测评后自动建档：客户完成六维测评 → 自动创建客户档案
- 带看后一键记录：生成带看报告 → 点击"写跟进" → 自动带入信息
- 解码后标记状态：使用客户解码器 → 收藏场景 → 标记"已用"

## 用户路径图（v1.1新增）

```
打开小程序 → 首页选工具 → 判断当前场景
  → 带看前获客 → 六维测评 → 分享海报
  → 客户说"不" → 客户解码器 → 收藏场景
  → 带看中 → 带看报告 → 生成PDF
  → 带看后跟进 → 客户跟进记录 → 设提醒
```

3分钟上手，场景驱动。

## 数据打通方案（v1.1新增）

| 数据类型 | 小程序端 | 网页端 | 同步方式 |
|----------|----------|--------|----------|
| 测评结果 | 生成海报、查看历史 | 查看完整报告、对比分析 | 实时同步（D1） |
| 收藏/场景 | 收藏解码场景 | 收藏知识文章 | 实时同步 |
| 客户档案 | 查看+编辑（独家） | 只读查看 | 实时同步 |
| 跟进记录 | 写跟进+设提醒（独家） | 只读查看 | 实时同步 |
| 用户身份 | 微信一键登录 | 手机号/微信扫码登录 | 统一unionId |

## 技术方案

- 技术选型：微信原生小程序（推荐，团队小不值得跨端）
- 后端：复用现有 Cloudflare Workers + D1 数据库
- 架构：小程序前端 → Cloudflare API → D1 → 管理后台

## 核心定位

网页版 vs 小程序版差异化：

- 网页版 = 展示 + 行业背书 + 机构合作（面向行业/高校/B端）
- 小程序版 = 随身工具 + 高频使用 + 个人赋能（面向一线经纪人/服务者）

小程序不是网页的缩小版，而是独立的产品形态。

## 功能架构

### 首页：卡片式工具入口

- 六维品质测评（最大卡片，获客入口）
- 客户解码器 + 带看报告（并排）
- 关怀测评 + AI助手（并排）
- 知识底座（底部入口）

### 5个核心功能

1. 六维品质测评：精简到10题1分钟 + 海报分享 + 客户管理 + 批量测评
2. 客户解码器：最近使用置顶 + 搜索 + 语音输入 + 收藏 + 分享
3. 带看报告：拍照+语音录入 + PDF导出 + 房源库 + 带看日历
4. AI助手（小程序独有）：语音对话 + 场景推荐 + 每日一学 + 随身顾问
5. 个人中心：数据统计 + 收藏 + 客户管理 + 专属名片 + 设置

## 技术方案

- 技术选型：微信原生小程序（推荐）
- 后端：复用现有 Cloudflare Workers + D1 数据库
- 架构：小程序前端 → Cloudflare API → D1 → 管理后台

## MVP交付计划

第一期（2-3周）：六维品质测评 + 客户解码器 + 知识底座 + 个人中心
第二期（+2周）：带看报告 + AI语音助手
第三期（+2周）：客户管理 + 分享裂变 + 团队版

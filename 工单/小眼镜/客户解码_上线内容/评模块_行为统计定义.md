# 客户解码 · 评模块行为统计3项定义

> **版本**：v1.0 | 2026.06.30 | 小眼镜·教培研发
> **功能**：定义3项行为数据的采集方式、存储结构和前端展示
> **决策依据**：MVP决策D8（仅3项基础统计，4维AI使用习惯画像V2再做）
> **存储方式**：localStorage，不做后端数据库

---

## 一、3项行为数据总览

| # | 统计项 | 数据来源 | 采集时机 | 采集方式 | 存储位置 |
|---|--------|---------|---------|---------|---------|
| 1 | 使用次数 | 每次完成解码练习 | 点击"保存到成长档案"时 | +1 | `behaviorStats.totalUsage` |
| 2 | 场景类型分布 | 每次练习记录场景ID | 点击"保存到成长档案"时 | 按 sceneId +1 | `behaviorStats.sceneDistribution` |
| 3 | 解码满意度 | 每次练习后1-5星评分 | 解码完成后弹出评分 | 追加评分值 | `behaviorStats.satisfactionScores` |

## 二、采集方式详细定义

### 2.1 使用次数

| 属性 | 说明 |
|------|------|
| **定义** | 累计使用解码器的总次数 |
| **触发条件** | 点击"保存到成长档案"按钮时 +1 |
| **统计维度** | 总计 + 按日统计 + 按场景统计 + 按线路统计 |
| **防重复** | 同一次解码只允许保存一次（按钮保存后变为"已保存"禁用态） |
| **清零规则** | 不清零，持续累计 |

**采集代码**：
```javascript
function incrementUsage(scenarioId, line) {
  const stats = JSON.parse(localStorage.getItem('behaviorStats') || '{}');

  // 总次数
  stats.totalUsage = (stats.totalUsage || 0) + 1;

  // 按日统计
  const today = new Date().toISOString().split('T')[0]; // "2026-06-30"
  stats.dailyUsage = stats.dailyUsage || {};
  stats.dailyUsage[today] = (stats.dailyUsage[today] || 0) + 1;

  // 按场景统计
  stats.sceneDistribution = stats.sceneDistribution || {};
  stats.sceneDistribution[scenarioId] = (stats.sceneDistribution[scenarioId] || 0) + 1;

  // 按线路统计
  stats.lineDistribution = stats.lineDistribution || {};
  stats.lineDistribution[line] = (stats.lineDistribution[line] || 0) + 1;

  localStorage.setItem('behaviorStats', JSON.stringify(stats));
}
```

### 2.2 场景类型分布

| 属性 | 说明 |
|------|------|
| **定义** | 各场景完成次数，交易vs租赁占比 |
| **触发条件** | 同"使用次数"，在保存时一并记录 |
| **统计维度** | 按场景ID（T1-T5/R1-R5）× 按线路（trade/rental） |
| **计算方式** | 直接从 `sceneDistribution` 和 `lineDistribution` 读取 |

**场景ID→名称映射**：
```javascript
const SCENE_MAP = {
  T1: '首次置业·刚需上车',  T2: '改善换房·升级置换',
  T3: '子女教育·学区购房',  T4: '婚房购置·甜蜜刚需',
  T5: '法拍购房·捡漏心态',  R1: '工作便利·通勤优先',
  R2: '投资出租·房东收益',  R3: '租赁过渡·先租后买',
  R4: '养老安住·适老优先',  R5: '品质升级·终极改善'
};
```

**线路占比计算**：
```javascript
function getLineRatio() {
  const stats = JSON.parse(localStorage.getItem('behaviorStats') || '{}');
  const trade = (stats.lineDistribution?.trade || 0);
  const rental = (stats.lineDistribution?.rental || 0);
  const total = trade + rental;
  if (total === 0) return { trade: '0%', rental: '0%' };
  return {
    trade: Math.round(trade / total * 100) + '%',
    rental: Math.round(rental / total * 100) + '%'
  };
}
```

### 2.3 解码满意度

| 属性 | 说明 |
|------|------|
| **定义** | 每次练习后对解码结果的自评满意度（1-5星） |
| **触发条件** | 解码完成后弹出评分弹窗（可选，可跳过） |
| **评分范围** | 1星（很不满意）→ 5星（非常满意） |
| **统计方式** | 追加到数组 + 计算平均分 |
| **可选性** | 用户可跳过评分，跳过则不记录 |

**评分弹窗交互**：
```
解码报告展示完成后，底部弹出：
┌─────────────────────────────────┐
│  这次解码对你有帮助吗？            │
│                                 │
│  ⭐ ⭐ ⭐ ⭐ ⭐                  │
│  1   2   3   4   5             │
│                                 │
│  [跳过]           [提交评分]      │
└─────────────────────────────────┘
```

**采集代码**：
```javascript
function submitSatisfaction(score) {
  // score: 1-5，整数
  if (score < 1 || score > 5) return;

  const stats = JSON.parse(localStorage.getItem('behaviorStats') || '{}');

  // 追加评分
  stats.satisfactionScores = stats.satisfactionScores || [];
  stats.satisfactionScores.push(score);

  // 计算平均分（保留1位小数）
  const sum = stats.satisfactionScores.reduce((a, b) => a + b, 0);
  stats.averageSatisfaction = Math.round(sum / stats.satisfactionScores.length * 10) / 10;

  localStorage.setItem('behaviorStats', JSON.stringify(stats));
}
```

### 3.2 字段定义

| 字段路径 | 类型 | 说明 | 初始值 |
|---------|------|------|--------|
| `behaviorStats` | Object | 行为统计根对象 | `{}` |
| `behaviorStats.totalUsage` | Number | 累计使用总次数 | `0` |
| `behaviorStats.dailyUsage` | Object\<string, number\> | 按日使用次数，key=日期字符串 | `{}` |
| `behaviorStats.dailyUsage["2026-06-30"]` | Number | 某日使用次数 | — |
| `behaviorStats.sceneDistribution` | Object\<string, number\> | 按场景使用次数，key=场景ID | `{}` |
| `behaviorStats.sceneDistribution["T1"]` | Number | 某场景使用次数 | — |
| `behaviorStats.lineDistribution` | Object\<string, number\> | 按线路使用次数 | `{}` |
| `behaviorStats.lineDistribution.transaction` | Number | 交易线使用次数 | `0` |
| `behaviorStats.lineDistribution.rental` | Number | 租赁线使用次数 | `0` |
| `behaviorStats.satisfactionScores` | Array\<number\> | 历史评分列表 | `[]` |
| `behaviorStats.averageSatisfaction` | Number | 平均满意度（1位小数） | `0` |

### 3.3 容量预估

| 项目 | 单条大小 | 条数上限 | 预估总量 |
|------|---------|---------|---------|
| totalUsage | 8B | 1 | 8B |
| dailyUsage | ~25B/条 | 365条 | ~9KB |
| sceneDistribution | ~15B/条 | 10条 | ~150B |
| lineDistribution | ~30B/条 | 2条 | ~60B |
| satisfactionScores | 4B/条 | 500条 | ~2KB |
| averageSatisfaction | 8B | 1 | 8B |
| **合计** | — | — | **~12KB** |

localStorage上限5MB，预估12KB远低于上限，无需担心容量问题。

### 4.2 三项指标卡片

#### 卡片1：使用次数

| 元素 | 内容 | 数据源 |
|------|------|--------|
| 主数字 | `18` | `behaviorStats.totalUsage` |
| 单位 | `次` | 固定文案 |
| 副文字 | `近7天: 18次` | 从 `dailyUsage` 计算最近7天总和 |

**近7天计算**：
```javascript
function getRecent7DaysUsage() {
  const stats = JSON.parse(localStorage.getItem('behaviorStats') || '{}');
  const daily = stats.dailyUsage || {};
  const today = new Date();
  let sum = 0;
  for (let i = 0; i < 7; i++) {
    const d = new Date(today);
    d.setDate(d.getDate() - i);
    const key = d.toISOString().split('T')[0];
    sum += daily[key] || 0;
  }
  return sum;
}
```

#### 卡片2：场景类型分布

| 元素 | 内容 | 数据源 |
|------|------|--------|
| 主展示 | `交易:10 / 租赁:8` | `lineDistribution` |
| 占比条 | 交易55%/租赁45% | 计算得出 |
| 场景覆盖 | `已覆盖 6/10 场景` | `Object.keys(sceneDistribution).length` |

**场景覆盖计算**：
```javascript
function getSceneCoverage() {
  const stats = JSON.parse(localStorage.getItem('behaviorStats') || '{}');
  const covered = Object.keys(stats.sceneDistribution || {}).length;
  return `${covered}/10`;

```

#### 卡片3：解码满意度

| 元素 | 内容 | 数据源 |
|------|------|--------|
| 主数字 | `4.1` | `averageSatisfaction` |
| 星级 | `⭐⭐⭐⭐☆` | 根据 `averageSatisfaction` 渲染 |
| 评分次数 | `12次评分` | `satisfactionScores.length` |

**星级渲染**：
```javascript
function renderStars(avg) {
  const full = Math.floor(avg);
  const half = avg - full >= 0.5;
  const empty = 5 - full - (half ? 1 : 0);
  return '★'.repeat(full) + (half ? '☆' : '') + '☆'.repeat(empty);
}
```

## 五、数据读取封装

### 5.1 统一读写接口

```javascript
const BehaviorStats = {
  /**
   * 获取全部统计数据
   */
  getAll() {
    return JSON.parse(localStorage.getItem('behaviorStats') || '{}');
  },

  /**
   * 保存全部统计数据
   */
  save(data) {
    localStorage.setItem('behaviorStats', JSON.stringify(data));
  },

  /**
   * 记录一次使用
   */
  recordUsage(scenarioId, line) {
    const stats = this.getAll();
    stats.totalUsage = (stats.totalUsage || 0) + 1;

    const today = new Date().toISOString().split('T')[0];
    stats.dailyUsage = stats.dailyUsage || {};
    stats.dailyUsage[today] = (stats.dailyUsage[today] || 0) + 1;

    stats.sceneDistribution = stats.sceneDistribution || {};
    stats.sceneDistribution[scenarioId] = (stats.sceneDistribution[scenarioId] || 0) + 1;

    stats.lineDistribution = stats.lineDistribution || {};
    stats.lineDistribution[line] = (stats.lineDistribution[line] || 0) + 1;

    this.save(stats);
  },

  /**
   * 记录满意度评分
   */
  recordSatisfaction(score) {
    if (score < 1 || score > 5) return;
    const stats = this.getAll();
    stats.satisfactionScores = stats.satisfactionScores || [];
    stats.satisfactionScores.push(score);
    const sum = stats.satisfactionScores.reduce((a, b) => a + b, 0);
    stats.averageSatisfaction = Math.round(sum / stats.satisfactionScores.length * 10) / 10;
    this.save(stats);
  },

  /**
   * 获取使用次数
   */
  getTotalUsage() {
    return this.getAll().totalUsage || 0;
  },

  /**
   * 获取线路占比
   */
  getLineRatio() {
    const stats = this.getAll();
    const trade = stats.lineDistribution?.transaction || 0;
    const rental = stats.lineDistribution?.rental || 0;
    return { transaction: trade, rental: rental };
  },

  /**
   * 获取场景覆盖数
   */
  getSceneCoverage() {
    const stats = this.getAll();
    return Object.keys(stats.sceneDistribution || {}).length;
  },

  /**
   * 获取平均满意度
   */
  getAverageSatisfaction() {
    return this.getAll().averageSatisfaction || 0;
  },

  /**
   * 获取近7日数据
   */
  getRecent7Days() {
    const stats = this.getAll();
    const daily = stats.dailyUsage || {};
    const today = new Date();
    const result = [];
    for (let i = 6; i >= 0; i--) {
      const d = new Date(today);
      d.setDate(d.getDate() - i);
      const key = d.toISOString().split('T')[0];
      result.push({
        date: `${d.getMonth()+1}/${d.getDate()}`,
        count: daily[key] || 0
      });
    }
    return result;
  },

  /**
   * 获取场景排行
   */
  getSceneRanking() {
    const stats = this.getAll();
    const dist = stats.sceneDistribution || {};
    return Object.entries(dist)
      .map(([id, count]) => ({ id, name: SCENE_MAP[id] || id, count }))
      .sort((a, b) => b.count - a.count);
  }
};
```

### 5.2 初始化函数

```javascript
/**

function initBehaviorStats() {
  if (!localStorage.getItem('behaviorStats')) {
    const initial = {
      totalUsage: 0,

      sceneDistribution: {},
      lineDistribution: { transaction: 0, rental: 0 },

  }
}
```

## 六、与评模块其他组件的联动

| 触发点 | 行为统计更新 | 成长档案更新 | 说明 |
|--------|------------|------------|------|
| 点击"保存到成长档案" | `totalUsage+1` + 场景/线路分布更新 | 追加解码历史+摘要复制 | 同一次操作触发两个更新 |
| 提交满意度评分 | 追加评分+更新平均分 | 无 | 独立操作，不影响档案 |
| 跳过满意度评分 | 无更新 | 无 | 可选操作，跳过不记录 |
| 打开成长档案页面 | 无更新 | 读取 `behaviorStats` 渲染统计面板 | 仅读取不写入 |

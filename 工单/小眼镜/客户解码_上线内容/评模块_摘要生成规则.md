# 客户解码 · 评模块"保存到成长档案"摘要生成规则

> **版本**：v1.0 | 2026.06.30 | 小眼镜·教培研发
> **功能**：定义从解码报告到摘要文本的生成逻辑
> **决策依据**：MVP决策D3（手动触发导出，不做自动回流）
> **前端实现**：点击"保存到成长档案"→生成摘要→复制到剪贴板

---

## 一、摘要结构定义

### 1.1 完整摘要模板

```
📋 解码摘要 | {date}
场景：{scenarioName} | 档位：{gradeLabel}
🔍 显性需求：{explicitItems}
💡 隐性发现：{implicitItems}
⭐ 关键洞察：Top1关注+权重
📝 我的话术：（学员自填）
```

### 1.2 字段说明

| 字段 | 来源 | 生成方式 | 长度限制 |
|------|------|---------|---------|
| `{date}` | `new Date().toLocaleDateString('zh-CN')` | 直接提取系统日期 | 10字符（2026/6/30） |
| `{scenarioName}` | 场景数据 `scenario.name` | 直接提取场景名称 | ≤20字符 |
| `{gradeLabel}` | localStorage `grade.current` | 直接提取档位标签（L0本能型/L1标准型/L2专业型） | ≤8字符 |
| `{explicitItems}` | X3输出 `layer1_explicit.items[]` | **拼接**：取前3条content，用"·"连接 | ≤60字符 |
| `{implicitItems}` | X3输出 `layer2_implicit.items[]` | **拼接**：取前2条content，用"；"连接 | ≤80字符 |
| Top1关注+权重 | X3输出 `layer3_priority.top3[0]` | **拼接**：`top3[0].concern + "(" + top3[0].weight + "%)"` | ≤20字符 |
| 我的话术 | 学员手动输入 | **自填**：文本框输入，非自动生成 | ≤200字符 |

### 1.3 生成规则详解

| 规则类别 | 字段 | 规则 |
|---------|------|------|
| **直接提取** | date | `new Date().toLocaleDateString('zh-CN')`，格式：yyyy/M/d |
| **直接提取** | scenarioName | 从当前练习场景的JSON数据中读取 `scenario.name` |
| **直接提取** | gradeLabel | 从localStorage读取 `grade.current`，映射为"Ln + 档位名" |
| **截取拼接** | explicitItems | 取 `layer1_explicit.items` 前3条的 `content`，用"·"连接；超3条省略并加"等" |
| **截取拼接** | implicitItems | 取 `layer2_implicit.items` 前2条的 `content`，用"；"连接；超2条省略并加"等" |
| **组合拼接** | Top1关注+权重 | `top3[0].concern` + `"("` + `top3[0].weight` + `"%)"` |
| **自填** | 我的话术 | 空白文本框，学员手动输入，非必填 |

## 二、摘要生成函数（JS伪代码）

```javascript
/**
 * 生成解码摘要文本
 * @param {Object} decodeOutput - 解码器三层输出（X3 Schema）
 * @param {Object} scenarioData - 当前场景数据
 * @returns {Object} { summary: string, plainText: string }
 */
function generateDecodeSummary(decodeOutput, scenarioData) {
  // ━━━ 1. 日期 ━━━
  const date = new Date().toLocaleDateString('zh-CN'); // "2026/6/30"

  // ━━━ 2. 场景名称 ━━━
  const scenarioName = scenarioData.name; // "首次置业·刚需上车"

  // ━━━ 3. 档位标签 ━━━
  const gradeLabel = getGradeLabel(); // "L1标准型"
  function getGradeLabel() {
    const gradeMap = { L0: 'L0本能型', L1: 'L1标准型', L2: 'L2专业型' };
    const current = JSON.parse(localStorage.getItem('userProfile') || '{}')
      ?.grade?.current || 'L0';
    return gradeMap[current] || 'L0本能型';
  }

  // ━━━ 4. 显性需求（取前3条，用"·"连接）━━━
  const explicitItems = decodeOutput.layer1_explicit.items
    .slice(0, 3)
    .map(item => item.content)
    .join('·');
  // 例："预算150-300万·首次购房·望京区域"
  const explicitOverflow = decodeOutput.layer1_explicit.items.length > 3;
  const explicitDisplay = explicitOverflow
    ? explicitItems + '等'
    : explicitItems;

  // ━━━ 5. 隐性发现（取前2条，用"；"连接）━━━
  const implicitItems = decodeOutput.layer2_implicit.items
    .slice(0, 2)
    .map(item => item.content)
    .join('；');
  // 例："预算有弹性，可上浮至320万看性价比；男女关注点不同需分别沟通"
  const implicitOverflow = decodeOutput.layer2_implicit.items.length > 2;
  const implicitDisplay = implicitOverflow
    ? implicitItems + '等'
    : implicitItems;

  // ━━━ 6. Top1关注+权重 ━━━
  const top1 = decodeOutput.layer3_priority.top3[0];
  const topInsight = `${top1.concern}(${top1.weight}%)`;
  // 例："通勤便利(35%)"

  // ━━━ 7. 组装摘要 ━━━
  const summary = [
    `📋 解码摘要 | ${date}`,
    `场景：${scenarioName} | 档位：${gradeLabel}`,
    `🔍 显性需求：${explicitDisplay}`,
    `💡 隐性发现：${implicitDisplay}`,
    `⭐ 关键洞察：${topInsight}`,
    `📝 我的话术：（学员自填）`
  ].join('\n');

  // ━━━ 8. 纯文本版本（用于剪贴板，去除emoji）━━━
  const plainText = summary
    .replace(/📋|🔍|💡|⭐|📝/g, '')
    .replace('（学员自填）', '');

  return { summary, plainText };
}
```

### 2.1 长度限制函数

```javascript
/**
 * 截断过长文本，确保摘要总长度≤500字符
 * @param {string} text - 待截断文本
 * @param {number} maxLen - 最大字符数
 * @returns {string} 截断后文本
 */
function truncateText(text, maxLen) {
  if (text.length <= maxLen) return text;
  return text.slice(0, maxLen - 1) + '…';
}

// 各字段长度控制
const MAX_EXPLICIT = 60;   // 显性需求≤60字符
```

### 3.2 前端实现代码（核心逻辑）

```javascript
/**
 * "保存到成长档案"按钮点击处理
 */
async function handleSaveToGrowthFile() {
  // 1. 获取当前解码输出和场景数据
  const decodeOutput = getCurrentDecodeOutput();  // 解码器三层输出
  const scenarioData = getCurrentScenarioData();  // 当前场景JSON

  // 2. 生成摘要
  const { summary, plainText } = generateDecodeSummary(decodeOutput, scenarioData);

  // 3. 弹出预览弹窗（含"我的话术"输入框）
  const userScript = await showSummaryPreviewModal(summary);
  // userScript = 用户填写的话术文本，可能为空字符串

  // 4. 组装最终摘要（替换自填区域）
  const finalSummary = summary.replace(
    '📝 我的话术：（学员自填）',
    `📝 我的话术：${userScript || '（未填写）'}`
  );

  // 5. 复制到剪贴板
  try {
    await navigator.clipboard.writeText(finalSummary);
    showSuccessToast('✅ 摘要已复制，请粘贴到成长档案-学习记录');
  } catch (err) {
    // 降级方案：创建临时textarea
    fallbackCopyToClipboard(finalSummary);
    showSuccessToast('✅ 摘要已复制，请粘贴到成长档案-学习记录');
  }

  // 6. 更新行为统计
  updateBehaviorStats(scenarioData);

  // 7. 追加到解码历史
  appendDecodeHistory({
    date: new Date().toISOString(),
    scenarioId: scenarioData.id,
    scenarioName: scenarioData.name,
    summary: finalSummary,
    decisionScore: decodeOutput.decisionScore.value
  });
}

/**
 * 降级复制方案（兼容不支持clipboard API的浏览器）
 */
function fallbackCopyToClipboard(text) {
  const textarea = document.createElement('textarea');
  textarea.value = text;
  textarea.style.position = 'fixed';
  textarea.style.left = '-9999px';
  document.body.appendChild(textarea);
  textarea.select();
  document.execCommand('copy');
  document.body.removeChild(textarea);
}

/**
 * 更新行为统计
 */
function updateBehaviorStats(scenarioData) {
  const stats = JSON.parse(localStorage.getItem('behaviorStats') || '{}');

  // 使用次数+1
  stats.totalUsage = (stats.totalUsage || 0) + 1;

  // 按日统计
  const today = new Date().toISOString().split('T')[0]; // "2026-06-30"
  stats.dailyUsage = stats.dailyUsage || {};
  stats.dailyUsage[today] = (stats.dailyUsage[today] || 0) + 1;

  // 场景分布
  stats.sceneDistribution = stats.sceneDistribution || {};
  stats.sceneDistribution[scenarioData.id] =
    (stats.sceneDistribution[scenarioData.id] || 0) + 1;

  // 线路分布
  stats.lineDistribution = stats.lineDistribution || {};
  const line = scenarioData.line; // "trade" or "rental"
  stats.lineDistribution[line] = (stats.lineDistribution[line] || 0) + 1;

  localStorage.setItem('behaviorStats', JSON.stringify(stats));
}

/**
 * 追加解码历史（最近20条）
 */
function appendDecodeHistory(entry) {
  const history = JSON.parse(localStorage.getItem('decodeHistory') || '[]');
  history.unshift(entry); // 新记录在前
  if (history.length > 20) history.pop(); // 保留最近20条
  localStorage.setItem('decodeHistory', JSON.stringify(history));
}
```

## 五、边界情况处理

| 场景 | 处理方式 |
|------|---------|
| 解码输出为空（解码器未返回结果） | 弹窗提示"请先完成解码"，不弹出摘要预览 |
| 显性需求items为空 | 显示"🔍 显性需求：（无数据）" |
| 隐性发现items为空 | 显示"💡 隐性发现：（未识别到隐性需求）" |
| 优先级top3为空 | 显示"⭐ 关键洞察：（未生成优先级）" |
| "我的话术"未填写 | 替换为"（未填写）"，摘要仍可正常复制 |
| 剪贴板API不可用 | 降级到fallback复制方案 |
| 摘要总长度超500字符 | 各字段按长度限制截断，末尾加"…" |

## 一、设计原则 & 全景图

### 设计原则（6条 · 生哥定调）

| # | 原则 | 一句话 |
|---|------|--------|
| 1 | **定位** | 风声不做"教人干活"，做"教人不被替代" |
| 2 | **分工** | 碳基想明白「做什么」，硅基执行「怎么做」 |
| 3 | **课程内核** | 教服务者想明白事 + 用AI帮人做到位 |
| 4 | **节奏** | 不跳步：框架→大纲→课件→实操→考核 |
| 5 | **飞轮** | 以终为始：课程驱动用好工具→更好服务客户→飞轮转起来 |
| 6 | **入行门槛** | 新人先过职业素养关 — 先懂"为什么做服务者"，再学"怎么被客户认可" |

**底层骨架（贯穿全文，后文不再重复展开）：**

```
风声业务字典（做什么） + 人才能力字典（缺什么）
    → 5产品（工具能帮什么） + 职业素养（为什么做）
    → 通识课（底座） + 专业课（岗位） + 专项实训（验证）
```

### 三层架构

```
通识课（4模块 · 所有角色必修）
├── A4 职业素养入门    ← 新人第一关，4周
├── A1 AI能力培养      ← 核心引擎，8周
├── A2 自媒体IP打造     ← 增量，4周
└── A3 线上获客实战     ← 增量，4周

专业课（4岗位 · 按角色定制）
├── B1 经纪人           ← 最大用户群
├── B2 店长             ← 管理岗
├── B3 装修管家         ← 新职业
└── B4 客户经理         ← 去化核心，新增

专项实训（5线 · 产品验证闭环）
├── C1 Agent培养师      ← P0，≥30人
├── C2 客户解码器       ← P1，≥20人
├── C3 带看报告         ← P1，≥20人
├── C4 能力测评         ← P2，≥20人
└── C5 六维品质测评     ← P2，≥20人
```

### 三层关系

| 层级 | 定位 | 覆盖 | 产出 |
|------|------|------|------|
| **通识课** | 底座能力 | 所有角色必修 | 职业素养 + AI能力 + 增量能力 |
| **专业课** | 岗位能力 | 特定角色深化 | 按岗位的能力达标路径 |
| **专项实训** | 产品验证线 | 课程驱动用户实战 | 产品使用 → 反馈 → 验证计数 |

### 推荐学习路径

```
新人入行
  → A4 职业素养（懂为什么做）

  → B模块专业课（按岗位深化）

```
想明白时刻 → 方法论讲解 → 实操演练 → 产品衔接 → 验收标准

### A4：职业素养入门（4周）

**定位**：新人入行第一课。先懂"为什么做服务者"和"怎么被客户认可"，再学技能。

| 状态 | 框架 ✅ | 素材 ⚠️ | 初稿 ⬜ |
|------|---------|---------|---------|

**启动条件**：
- ✅ 设计原则已确认（生哥6.18定调）
- ✅ 四周框架已确认
- ⚠️ 素材来源待确认（开放问题#1）— 风声团队真实案例为主，是否需要外部素材

**设计依据**（对齐原则第6条）：
- 风声使命"帮服务者配得上被客户好好对待"——起点就是职业素养
- 职业素养是碳基独占能力（共情力/判断力/信任构建力），AI替代不了
- 服务五层模型中，第三到第五层（共情/判断/信任）是人的护城河

**阻塞项**：开放问题#1 — 风声团队真实案例 vs 外部素材？  
**预估工时**：素材确认后 4 天

| 周次 | 主题 | 核心能力 | 产品衔接 | 交付物 |
|------|------|----------|----------|--------|
| 第1周 | 行业认知与职业定位 | 理解行业价值链 + 自己的角色价值 | — | 职业定位卡 |
| 第2周 | 职业素养四基石 | 客户至上 / 诚实可信 / 合作共赢 / 拼搏进取 | 六维品质测评 | 职业素养自检表 |
| 第3周 | 服务力修炼 | 需求洞察 / 服务标准 / 超预期服务 | 客户解码器 | 服务场景实战卡 |
| 第4周 | 用AI加速融入 | 从新人 → 被客户认可的专业服务者 | 能力测评 | 个人成长计划 |

**每周设计要点：**

- **第1周**「行业认知」：让新人理解居住服务行业价值链，知道自己在哪一环。落实"配得上"对服务者意味着什么
- **第2周**「四基石」：用真实翻车案例说明 — 缺了哪一条，客户就不信任你。碳基思考：「你愿意把大事交给什么样的人？」
- **第3周**「服务力」：从"知道客户要什么"到"做到客户没说的"。用AI做客户场景模拟训练共情力
- **第4周**「用AI加速融入」：前置应用A1的AI能力 — 用AI写职业定位、模拟客户对话、做服务自检

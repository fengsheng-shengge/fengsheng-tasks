## 一、带看三备三核准备清单

### 1.1 三备（带看前准备）

#### 备房源

| 检查项 | 操作标准 | 完成打✓ |
|--------|----------|----------|
| 房屋信息熟记 | 面积、朝向、楼层、装修、家电品牌——至少能不看资料说清楚 | ☐ |
| 提前踩点 | 带看前24小时内到房源实地走一遍，确认：无异味、灯能亮、水能通、空调能开 | ☐ |
| 拍照/视频 | 拍摄≥6张照片（客厅/卧室/厨房/卫生间/阳台/窗外视野），≥1段视频（从进门到各房间完整走一遍） | ☐ |
| 周边信息 | 记录：最近地铁站步行时间、最近超市距离、附近餐饮数量、噪音情况（白天+晚上） | ☐ |
| 竞争对比 | 了解同小区同类房源当前租金、同价位可替代房源，做到心中有数 | ☐ |
| 房东情况 | 了解房东出租原因、是否接受议价、对租客要求（如禁养宠物等） | ☐ |

#### 备客户

| 检查项 | 操作标准 | 完成打✓ |
|--------|----------|----------|
| 基本画像 | 客户姓名/年龄/职业/预算/入住时间/同行人数 | ☐ |
| 核心需求 | 客户最在意什么？（通勤/价格/面积/装修/配套），排优先级 | ☐ |
| 决策人判断 | 谁拍板？如果是情侣/家庭，两个人的需求都要照顾到 | ☐ |
| 看房经历 | 之前看过哪些房子？为什么没定？（避开类似问题） | ☐ |
| 敏感点预判 | 根据客户画像预判可能的顾虑：噪音/采光/安全/价格 | ☐ |
| 替代方案 | 准备2-3套备选房源，万一这套不满意能马上切换 | ☐ |

#### 备话术

| 检查项 | 操作标准 | 完成打✓ |
|--------|----------|----------|
| 3个亮点话术 | 针对这套房子准备3个场景化亮点描述（用客户语言，不用行业术语） | ☐ |
| 2个抗性应对话术 | 预判客户可能提的2个问题，准备好回答（如"这层有点高"→"高层采光好、视野好、还安静"） | ☐ |
| 开场白 | 准备30秒自我介绍+本次带看路线说明 | ☐ |
| 收尾话术 | 准备结束时的2个引导选项（"要不要再看看另外一套"或"如果满意，咱们聊聊意向"） | ☐ |
| 价格铺垫 | 准备好谈价格时的锚点话术（"同小区这个户型平均XXX，这套因为XXX所以性价比很高"） | ☐ |

### 1.2 三核（带看中核验）

#### 核需求

| 核验点 | 怎么核 | 话术示例 |
|--------|--------|----------|
| 预算真实性 | 观察客户对价格的反应 | 「这套租金是XXX，您觉得在预算范围内吗？」 |
| 地段偏好 | 看客户对周边环境的评价 | 「这附近您觉得方便吗？上班通勤大概多久？」 |
| 户型偏好 | 注意客户在哪停留最久 | 「您觉得这个户型布局怎么样？厨房大小够用吗？」 |
| 隐性需求 | 听客户的"但是"和"如果" | 「您说'但是……'，是不是有什么不太满意的地方？」 |

#### 核匹配

| 核验点 | 怎么核 | 话术示例 |
|--------|--------|----------|
| 满意度 | 直接问打分 | 「如果1-10分，这套房子您打几分？」 |
| 对比参考 | 与其他看过的房源比较 | 「跟您之前看的XX小区比，您觉得这套怎么样？」 |
| 决策障碍 | 主动问顾虑 | 「如果要定这套，您最纠结的是什么？」 |

#### 核顾虑

| 核验点 | 怎么核 | 话术示例 |
|--------|--------|----------|
| 价格顾虑 | 观察是否反复问价格 | 「价格这块您觉得还合适吗？房东那边我可以帮您再谈谈。」 |
| 时间顾虑 | 确认入住时间是否匹配 | 「您下个月15号入住，时间上没问题吧？」 |
| 隐形顾虑 | 注意客户和同行人的交流 | 「您和（伴侣/朋友）商量一下，有哪里觉得不太对的地方尽管说。」 |

## 二、房屋介绍四必做话术模板

### 2.1 四必做框架

```
① 基础信息 → ② 亮点强化 → ③ 抗性铺垫 → ④ 价值总结
  "是什么"     "好在哪"      "别担心"      "为什么值"
```

### 2.2 每一步话术模板

#### ① 基础信息（30秒）

| 维度 | 内容 |
|------|------|
| **目的** | 让客户快速了解房子的「基本面」 |
| **必须说的** | 面积、户型、朝向、楼层、装修程度、租金、押付方式 |
| **话术模板** | 「这套是XX户型，面积XX平，朝X，在XX楼（共XX层）。XX年装修的，家电都是XX品牌的。月租XXX，押X付X，物业费XX/月已经含在租金里了。」 |
| **操作要点** | 语速适中，边说边指，让客户跟着你的节奏走。不要说太快——给客户留出观察的时间。 |
| **验收标准** | 7项基本信息完整覆盖，客户能复述其中≥5项 |

#### ② 亮点强化（2分钟）

| 维度 | 内容 |
|------|------|
| **目的** | 让客户记住3个「非选不可」的理由 |
| **话术模板** | 「这套房子我最喜欢的有三点——」<br>亮点1：「第一，你看这个窗户，朝南大窗，冬天阳光能晒到床的位置，早上醒来就暖洋洋的。省电又舒服。」<br>亮点2：「第二，厨房。你看这个台面、橱柜都是去年新换的，而且灶台位置靠窗，做饭的时候开窗通风，屋里不会有油烟味。」<br>亮点3：「第三，也是很多客户在意的——隔音。你听，外面马路的车声基本听不到，因为窗户是双层玻璃的。晚上睡觉很安静。」 |
| **操作要点** | ① 每个亮点用「你看」开头，引导客户亲眼看到；② 用具体场景（冬天晒太阳、做饭通风）而非抽象形容词；③ 每个亮点配一个「对比」，凸显优势 |
| **验收标准** | 客户能说出 ≥2个亮点（表示真的记住了） |

#### ③ 抗性铺垫（1分钟）

| 维度 | 内容 |
|------|------|
| **目的** | 主动说出可能的不足，提前化解客户顾虑，建立「这个顾问很诚实」的信任 |
| **话术模板** | 「我也实话跟您说，这套房子有两个地方需要注意——」<br>抗性1：「楼层稍微高了点，12楼，等电梯大概30秒。不过好处是视野好、采光好、还安静，楼下小孩玩闹的声音基本听不到。」<br>抗性2：「周边没有大型商场，最近的要坐两站地铁。不过小区楼下就有便利店和菜市场，日常买东西完全够用。」 |
| **操作要点** | ① 先承认不足（建立信任），再给积极解读（化解顾虑）；② 不超过2个抗性，多了显得房子问题大；③ 每个抗性都要有对应的「好处」 |
| **验收标准** | 客户没有就你说的问题追加质疑（说明你的化解有效） |

#### ④ 价值总结（30秒）

| 维度 | 内容 |
|------|------|
| **目的** | 用「对比」锚定价值，让客户觉得「值」 |
| **话术模板** | 「总结一下——同小区同户型现在租金普遍在XXX左右，这套因为XXX（亮点），性价比确实不错。而且房东着急租，价格还有空间。这个地段、这个装修，XXX的价格真的很难得。」 |
| **操作要点** | ① 至少用1个数据对比（同小区均价/上月成交价）；② 制造轻微稀缺感（"房东着急租""好几组在看"）；③ 不要过度推销——收在"您考虑一下" |
| **验收标准** | 客户点头或追问细节（表示对价值认可） |

## 三、好斡旋奇妙等式实操指南

### 3.1 核心公式

```
成交概率 = 意向度 × 紧迫度 - 抗性

其中：
- 意向度：1-10分，客户对这套房的喜欢程度
- 紧迫度：1-10分，客户有多急迫要成交
- 抗性：0-10分，客户有哪些顾虑/障碍
```

### 3.2 各要素评估表

#### 意向度评估（1-10分）

| 得分 | 判断信号 | 客户表现 |
|------|----------|----------|
| 1-3 | 低意向 | 看房时不停看手机、敷衍回应、不主动问问题、急着离开 |
| 4-6 | 中意向 | 会问一些细节问题、但也在挑毛病、没有明确表态 |
| 7-8 | 高意向 | 主动问价格/付款方式/入住时间、在某个区域停留很久、和同行人讨论 |
| 9-10 | 极高意向 | 直接问"怎么定"、"什么时候能签"、现场打电话和家人商量 |

#### 紧迫度评估（1-10分）

| 得分 | 判断信号 | 客户表现 |
|------|----------|----------|
| 1-3 | 不紧迫 | "随便看看"、"不着急"、没有明确入住时间 |
| 4-6 | 有一定紧迫 | 有大概入住时间但可以调整、正在比较多个选择 |
| 7-8 | 较紧迫 | 有明确入住时间（如月底到期）、已经在看第二三轮 |
| 9-10 | 非常紧迫 | 本周/下周必须入住、已经开始打包、原住处已退 |

#### 抗性识别与应对

| 抗性类型 | 识别信号 | 应对策略 |
|----------|----------|----------|
| **价格抗性** | "有点贵""超预算了""能不能便宜" | ① 算日均成本（月租÷30=日均XX元，强调性价比）；② 对比同小区均价；③ 谈付款方式弹性 |
| **地段抗性** | "有点远""通勤不太方便" | ① 实测通勤时间（拿出你提前测的数据）；② 强调安静/性价比/社区配套补偿 |
| **品质抗性** | "装修有点旧""家电不是新的" | ① 承认客观事实；② 强调功能完好+房东可配/可换；③ 对比"新装修可能有甲醛" |
| **决策抗性** | "再考虑考虑""跟家人商量一下" | ① 制造轻度紧迫感（"好几组在看"）；② 给具体商量时间（"明天中午前给我答复"）；③ 提供决策辅助材料（对比表/带看报告） |

### 3.3 让步幅度计算表

> **核心原则**：每次让步必须有对应收获，不做单向让步。

| 让步项目 | 让步幅度 | 需要换回什么 | 话术模板 |
|----------|----------|-------------|----------|
| **租金** | 月租降50-100 | 年付/半年付，或签长约（≥2年） | 「价格我可以跟房东谈谈，如果能年付的话，房东那边更容易松口。」 |
| **租金** | 月租降100-200 | 去掉部分家具/家电（客户自带） | 「如果您自己有洗衣机，这个可以搬走，租金能再少200。」 |
| **押金** | 押二付一→押一付一 | 签长约（≥2年）+ 良好信用证明 | 「押金可以谈，但需要您提供一下之前的租房记录或者工作证明，我跟房东沟通。」 |
| **入住时间** | 提前5-7天免租 | 年付或签长约 | 「如果您能年付，入住前这几天的租金可以免掉，等于省了XXX块。」 |
| **家具配置** | 加配洗衣机/冰箱 | 租金上浮50-100/月 | 「房东那边可以加配洗衣机，不过租金需要加100/月。您觉得需要吗？」 |
| **免租期** | 3-7天搬家期 | 签长约+按时交租承诺 | 「房东一般不给免租期，但我帮您争取3天搬家时间——前提是您能保证按时交租。」 |

### 3.4 斡旋实战流程

```
第1步：收集信息
├── 明确客户最高接受价
├── 明确房东最低接受价
├── 判断双方差距（差距≤5%有斡旋空间，>5%需要换房源）
└── 识别双方最在意的点（价格？时间？付款方式？）

第2步：制定策略
├── 从哪一方先谈？（通常先谈房东，拿到"底价"再跟客户谈）
├── 让步顺序：先让非价格项（时间/配置/付款方式），最后让价格
└── 底线设定：你最多能让多少？

第3步：执行斡旋
├── 跟房东：强调客户诚意+付款能力+长租意向
├── 跟客户：强调房源稀缺+价格优势+房东诚意
├── 关键：永远不要让双方直接谈——你就是缓冲带

第4步：促成成交
├── 差距缩到3%以内时，提出折中方案
├── 给双方一个"台阶"——各让一步
├── 限时决策："房东说今天之内给答复的话，这个价格可以"
```

## 四、二次带看三步走流程

### 4.1 流程图

```
分析原因 → 优化方案 → 重新邀约
   ↓           ↓           ↓
问出真相    精准调整    带新方案
(为什么没定) (找更合适的) (再次出击)
```

### 4.2 每一步操作标准

#### ① 分析原因（带看后24小时内）

| 维度 | 内容 |
|------|------|
| **操作标准** | ① 带看结束当天发消息感谢+提问；② 问题聚焦："您看完之后最顾虑什么？"（只问1个核心问题，不给选择题）；③ 记录客户的回答，判断属于哪类问题（价格/地段/品质/决策人） |
| **话术示例** | 「今天看的这套，您回去考虑得怎么样？说实话，您最纠结的点是什么？我帮您想想办法。」 |
| **常见错误** | ❌ 问"您觉得怎么样"→ 客户回"挺好的我再看看"→ 得不到真信息 |
| **验收标准** | 24小时内完成跟进，获取 ≥1个明确的未成交原因 |

#### ② 优化方案（获取原因后2小时内）

| 维度 | 内容 |
|------|------|
| **操作标准** | ① 根据原因分类处理：价格问题→找同区域更低价的替代房源或谈价空间；地段问题→推荐同价位更近的房源或实测通勤时间打消顾虑；品质问题→找装修更好/更新的替代房源；② 准备2-3套替代方案，每套说明"为什么更适合你" |
| **话术示例** | 「我理解，那个小区确实离地铁远了点。我帮您重新筛了一下——XX小区，同样户型，走到地铁只要3分钟，价格还便宜100。这是照片，您看看。」 |
| **常见错误** | ❌ 推荐跟上次一样的房源 → 客户觉得你不懂他 |
| **验收标准** | 2小时内给出 ≥2套替代方案，每套有明确"为什么更适合"的说明 |

#### ③ 重新邀约（给出方案后24小时内）

| 维度 | 内容 |
|------|------|
| **操作标准** | ① 直接邀约具体时间（不给"有空再约"的选项）；② 带新方案去（让客户感受到"上次说的你听进去了"）；③ 制造合理紧迫感（但不要骗人） |
| **话术示例** | 「这几套我觉得特别适合您，尤其是XX小区那套，刚上的，业主急租。明天下午3点您方便吗？我安排好路线，咱们一次看2-3套。」 |
| **常见错误** | ❌ "什么时候有空再约"→ 客户永远不会"有空" |
| **验收标准** | 二次邀约成功率 ≥50%（10个未成交客户中至少5个愿意再看） |

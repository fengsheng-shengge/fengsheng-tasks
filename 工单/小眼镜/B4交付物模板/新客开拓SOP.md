### 1.2 每一步操作标准与话术示例

#### ① 主动外呼

| 维度 | 内容 |
|------|------|
| **操作标准** | ① 名单筛选：按「区域+预算段+时间窗口」三维筛选，每次外呼批次≤30人，避免疲劳；② 话术准备：提前准备3套开场话术，根据不同来源渠道切换；③ 批量触达：单次通话≤3分钟，核心目标=获得「需求确认」而非当场成交 |
| **话术示例** | 「您好，我是XX区域的租赁顾问。看到您最近在关注这边的房子，想了解一下您是自住还是帮朋友看？我这边有几套刚上的房源，性价比不错，想看看合不合适。」 |
| **常见错误** | ❌ 上来就推房：「我这儿有一套特别好的房子，您要不要看看？」→ 客户无信任，秒挂 |
| **验收标准** | 每30通电话 ≥8个进入需求确认环节（转化率≥26%） |

#### ② 需求确认

| 维度 | 内容 |
|------|------|
| **操作标准** | 四问确认法——① 预算范围？（上限下限都问）② 目标地段？（具体到小区/地铁站）③ 入住时间？（紧迫度判断）④ 特殊需求？（宠物/楼层/朝向/合租）——最后做需求复述确认：「我理解一下，您的需求是……」 |
| **话术示例** | 「那我帮您梳理一下——您预算是3000-4000，希望离地铁走路不超过10分钟，下个月15号之前入住，不养宠物、要朝南、最好有电梯。是这样吗？」 |
| **常见错误** | ❌ 只问预算不追问：「您预算多少？」→ 客户说3000，实际可上浮到3500，损失匹配空间 |
| **验收标准** | 每个确认客户 ≥4个需求维度记录完整（预算/地段/时间/特殊需求） |

#### ③ 匹配推荐

| 维度 | 内容 |
|------|------|
| **操作标准** | ① 基于需求筛选房源库，按匹配度排序取TOP3；② 用「二选一法」推送：每次推2套，让客户有比较；③ 每套推送含：价格+面积+核心亮点（≤3个）+实拍图/视频 |
| **话术示例** | 「按您的需求，我挑了两套——A套在XX小区，3200/月，朝南带阳台，走路到地铁6分钟；B套在XX小区，3500/月，新装修首次出租，家电全配。您先看看照片，觉得哪套更感兴趣？」 |
| **常见错误** | ❌ 一次推5套以上 → 客户选择困难，回复率反而降低 |
| **验收标准** | 推荐-邀约转化率 ≥40%（推送后客户同意看房） |

#### ④ 邀约带看

| 维度 | 内容 |
|------|------|
| **操作标准** | ① 确认时间：给2个时间选项（"周三下午还是周四上午？"），不给开放式问题；② 发送地址：含精准导航链接+地标参照物（"小区门口有个便利店"）；③ 提醒准备：提前1天+提前2小时各提醒1次 |
| **话术示例** | 「那咱们约周三下午3点，在XX小区门口见。导航搜『XX小区南门』就行，门口有个红色招牌的便利店。您到之前5分钟给我发个消息。对了，记得带上身份证，看中的话可能当场要填意向。」 |
| **常见错误** | ❌ 只约不来：「好的，那到时候见」→ 没有给明确地址和参照物，客户找不到 |
| **验收标准** | 邀约-到场率 ≥70%（约了10个至少来7个） |

#### ⑤ 跟进维护

| 维度 | 内容 |
|------|------|
| **操作标准** | ① 带看当天：带看结束2小时内发感谢消息+房源总结；② 带看后第2天：跟进询问感受，推送替代方案（如不满意）；③ 带看后第7天：发一条市场信息/新上房源，保持存在感；④ 超过14天无回应：归档至「沉睡客户池」，每月1次激活触达 |
| **话术示例** | 第2天跟进：「昨天看的那两套，您回去考虑得怎么样？如果有哪里不太满意的，我帮您再找找类似的。」第7天：「最近这个小区又上了一个同户型，价格还便宜200，您要不要看看照片？」 |
| **常见错误** | ❌ 成交就不管 → 未成交客户是最大的浪费 |
| **验收标准** | 未成交客户7天内跟进率 ≥90%，14天内二次激活率 ≥30% |

## 二、站外获客五秘诀渠道清单

### 2.1 渠道总览

| 秘诀 | 适用阶段 | 投入成本 | 见效周期 | 推荐优先级 |
|------|----------|----------|----------|-----------|
| ① 社区深耕 | 入门期 | 低（时间） | 1-2个月 | ⭐⭐⭐⭐⭐ |
| ② 异业合作 | 入门-成长期 | 低 | 2-4周 | ⭐⭐⭐⭐ |
| ③ 内容输出 | 成长期 | 中（精力） | 1-3个月 | ⭐⭐⭐⭐ |
| ④ 老客转介绍 | 全阶段 | 极低 | 即时 | ⭐⭐⭐⭐⭐ |
| ⑤ 线上投放 | 进阶期 | 高（资金） | 1-2周 | ⭐⭐⭐ |

### 2.2 每渠道操作指南

#### ① 社区深耕

| 维度 | 操作指南 |
|------|----------|
| **目标** | 成为目标社区的「租赁顾问」，让邻居有问题就找你 |
| **操作步骤** | ① 选定1-2个目标社区（步行15分钟内），每周至少去3次；② 加入社区业主群/租户群，不发广告，只回答租赁相关问题；③ 主动认识保安/便利店老板/物业管家，建立信息网；④ 制作「社区租房攻略」小卡片，含周边配套、交通、租金参考 |
| **话术示例** | （业主群内）「看到有邻居问附近租金行情，我整理了一下这个小区近半年的一居室成交数据，平均3500，旺季能到4000，给大家参考。有出租需求也可以找我聊聊。」 |
| **验收标准** | 单社区月度获客 ≥5个，社区内转介绍率 ≥50% |

#### ② 异业合作

| 维度 | 操作指南 |
|------|----------|
| **目标** | 与社区周边商户建立互推关系，低成本获取客户 |
| **操作步骤** | ① 筛选合作对象：便利店/理发店/快递驿站/宠物店/健身房（选择客流量大、与租房场景相关的）；② 合作方式：你帮商户介绍租客，商户帮你推荐有租房需求的顾客；③ 建立利益机制：每成交1单给商户发红包/请吃饭；④ 定期维护：每月至少拜访1次，带杯奶茶 |
| **话术示例** | （对便利店老板）「老板，我专门做这附近的租赁，您这儿每天人来人往，要是碰到有想在这边租房子的，您跟我说一声。成了我请您吃饭。我这儿也放几张名片，方便您给顾客。」 |
| **验收标准** | 每合作商户月度转介绍 ≥2个有效客户 |

#### ③ 内容输出

| 维度 | 操作指南 |
|------|----------|
| **目标** | 通过有价值的内容建立专业形象，让客户主动找来 |
| **操作步骤** | ① 确定内容方向：租房攻略/区域分析/避坑指南/真实房源；② 选定平台：优先小红书（图文攻略）+ 抖音/视频号（房源短视频）；③ 内容频率：每周≥3条；④ 内容公式 = 痛点引入 + 干货分享 + 行动引导 |
| **话术示例** | （小红书标题）「在XX区租房，这3个小区千万别踩坑！」（正文）列出3个小区的问题（隔音差/物业差/周边噪音），给出替代推荐，结尾「想知道更多区域选房技巧，关注我，下期讲XX区」 |
| **验收标准** | 月度内容产出 ≥12条，内容获客 ≥3个/月 |

#### ④ 老客转介绍

| 维度 | 操作指南 |
|------|----------|
| **目标** | 让每一个成交客户成为你的「获客节点」 |
| **操作步骤** | ① 成交时种种子：「您朋友要租房记得找我」；② 入住1周后回访，再次提转介绍；③ 设置转介绍激励机制：介绍成功送保洁/搬家券/红包；④ 每月梳理「高转介绍客户」名单，重点维护 |
| **话术示例** | 「对了，您身边要是也有朋友要租房，随时推给我。我帮您朋友找，也帮您把关。介绍成功了，我送您一次深度保洁，搬家也省心。」 |
| **验收标准** | 单客户转介绍率 ≥30%，转介绍成交率 ≥50% |

#### ⑤ 线上投放

| 维度 | 操作指南 |
|------|----------|
| **目标** | 通过付费投放规模化获取精准客户 |
| **操作步骤** | ① 平台选择：抖音（信息流广告）+ 小红书（薯条推广）；② 素材准备：15-30秒房源短视频，突出亮点+价格锚点；③ 投放设置：地域定向（目标社区周边3km）+ 兴趣标签（租房/搬家/应届生）；④ 数据追踪：每日记录曝光-点击-私信-加微信-带看转化数据 |
| **话术示例** | （广告文案）「XX地铁站步行5分钟，朝南带阳台，月租2800。这周六统一看房，私信我预约。」 |
| **验收标准** | 单客户获客成本 ≤50元，ROI（成交佣金/投放成本）≥5倍 |

## 三、房源传递四步走话术库

### 3.1 流程图

```
信息整理 → 价值包装 → 精准推送 → 反馈跟进
   ↓            ↓           ↓           ↓
房源档案    房源介绍     定向发送    持续改进
(标准化)   (客户语言)   (对的人)    (优化下一轮)
```

### 3.2 每一步操作标准与话术

#### ① 信息整理——建立房源档案

| 维度 | 内容 |
|------|------|
| **操作标准** | 每套房源建档含6大模块：基础信息（面积/户型/楼层/朝向/装修）、价格信息（租金/押金/付款方式/含不含物业）、配套信息（地铁/超市/医院/学校）、亮点标签（朝南/新装修/带阳台/近地铁）、实拍素材（照片≥6张+视频≥1段）、竞争对比（同小区/同价位对比） |
| **话术示例** | （内部档案格式，非对外）「XX小区·2居室｜面积68㎡｜楼层12/18｜朝南｜精装｜月租3500含物业｜地铁A口步行8分钟｜亮点：新装修首次出租、全屋品牌家电、带独立阳台」 |
| **验收标准** | 每套房源档案6大模块完整度 = 100% |

#### ② 价值包装——用客户语言描述

| 维度 | 内容 |
|------|------|
| **操作标准** | ① 提炼3个核心卖点，用「客户语言」而非「房源术语」描述；② 每个卖点配1句「场景化描述」，让客户想象住在里面的画面；③ 定价锚点：对比同小区均价，突出性价比 |
| **话术示例** | 「这套房子的三个亮点——第一，朝南大窗户，冬天阳光能晒到床上，不用开空调也暖和；第二，厨房刚翻新过，橱柜都是新的，爱做饭的话会很享受；第三，出门左转200米就是菜市场，下班顺路就能买菜，不用专门跑超市。」 |
| **常见错误** | ❌ 「全明户型、南北通透、动静分离」→ 客户听不懂行业术语 |
| **验收标准** | 每条推送含 ≥3个场景化卖点 + ≥1个定价锚点 |

#### ③ 精准推送——推给对的人

| 维度 | 内容 |
|------|------|
| **操作标准** | ① 按客户标签匹配房源：预算匹配度≥80%、地段匹配、户型匹配；② 推送时机：工作日19:00-21:00，周末10:00-12:00（客户看手机高峰）；③ 推送格式：文字介绍 + 3-6张精选图 + 1段短视频 |
| **话术示例** | 「上次您说想要朝南、预算3000左右的，今天刚上一套——XX小区，朝南大窗，月租2800，走路到地铁5分钟。这是照片，您先看看。这周末有两组客户约看，有兴趣的话我帮您优先排时间。」 |
| **常见错误** | ❌ 群发同一条消息 → 客户感受不到「被重视」 |
| **验收标准** | 推送-回复率 ≥60%，推送-带看率 ≥25% |

#### ④ 反馈跟进——持续优化

| 维度 | 内容 |
|------|------|
| **操作标准** | ① 每次推送后记录客户反馈（感兴趣/不感兴趣/原因）；② 根据反馈调整推送策略：不感兴趣的原因→下次避开类似房源；③ 每周做1次推送效果复盘：哪种房源回复率高？哪个时间段效果好？ |
| **话术示例** | （收集反馈）「这套您觉得怎么样？是价格不合适还是地段不太理想？您直说，我下次帮您挑得更准。」 |
| **验收标准** | 每周推送复盘 ≥1次，推送准确率逐周提升 |

## 一、方法论

### 1.1 业务字典与人才字典的关系

```
业务字典 → 定义行业知识边界（域→子域→词条）
人才字典 → 定义谁该掌握这些知识（岗位→等级→能力标准）
课程体系 → 教怎么掌握 → 认证 → 证明掌握了
```

### 1.2 域→岗位对应逻辑

每个业务域的知识边界决定需要什么岗位来执行。岗位定义遵循三方交叉验证：

| 验证方 | 视角 | 验证什么 |
|--------|------|----------|
| **国家标准** | 顶层定义 | 职业编码、技能等级、认证要求 |
| **行业实践** | 执行标准 | 企业岗位体系、培训路径、工具方法 |
| **消费者感知** | 需求反馈 | 信任要素、痛点、服务期待 |

## 三、域1·房产交易 → 岗位标准

### A1 房产经纪人

> **域归属：** 域1·房产交易 | **国标：** ✅ 4-06-02-01（房地产经纪人）
> **别名：** 置业顾问、房产顾问、物业顾问
> **十五五优先级：** 🟠高 — 存量交易+租赁规范化驱动技能转型

---

#### 初级（五级）· 实习经纪人

**岗位定义：** 在指导下完成房源录入、客户带看、合同签署等基础经纪业务，确保信息真实、流程合规。

**学历与经验：**
- 高中以上学历
- 通过房地产经纪从业人员资格考试
- 入职培训不少于40学时

**核心能力：**

| 维度 | 具体要求 |
|------|----------|
| **房源管理** | 能独立完成房源实地勘验，准确填写《房屋状况说明书》所列全部条目（面积/户型/朝向/楼层/装修/权属）；识别房屋权属类型（商品房/经适房/公房/小产权）并说明交易限制 |
| **客户接待** | 能按标准流程完成客户首次接待：①了解购房需求（预算/区域/户型/时间）②介绍服务流程与佣金标准 ③签订《房地产经纪服务合同》 |
| **带看执行** | 能完成标准带看流程：提前24小时确认时间→准备房源资料包（户型图/小区信息/周边配套/税费估算）→带看中记录客户反馈→带看后24小时内回访 |
| **合同签署** | 能识别买卖合同/租赁合同/居间合同三类合同的使用场景，在指导下正确填写标准合同模板中的标的物信息、交易价格、付款方式、交房日期 |
| **法规基础** | 熟记《房地产经纪管理办法》中关于禁止行为的14条规定；能说明资金监管的必要性和操作流程 |

**典型工作产出：**
- 房屋状况说明书（每套录入房源1份）
- 客户需求记录表
- 带看反馈记录
- 已签署的经纪服务合同

**AI工具要求：**
- 能使用AI找房工具完成房源初步匹配
- 能用AI生成带看报告初稿
- 能操作房产信息管理系统（房源录入/状态更新）

**认证方式：** 全国房地产经纪从业人员资格考试

---

#### 中级（四级）· 执业经纪人

**岗位定义：** 独立负责房源开发、客户维护、交易谈判全流程，能处理常见交易风险，完成从委托到过户的完整闭环。

**学历与经验：**
- 大专以上学历（或同等学力）
- 2年以上从业经验，累计完成交易不少于20单
- 通过全国房地产经纪人职业资格考试

**核心能力：**

| 维度 | 具体要求 |
|------|----------|
| **房源开发** | 能运用社区精耕、网络营销、老客转介、异业合作四种渠道独立开发房源；每月新增有效房源不少于5套；能对房源进行竞争力分析（同小区挂牌量/成交周期/价格走势） |
| **客户经营** | 能建立并维护200人以上有效客户池，按A（1月内决策）/B（3月内）/C（观望）分级管理；对A类客户保持每周1次有效沟通 |
| **交易谈判** | 能运用议价五步法：①建立价格锚点（同小区成交价+评估价）②挖掘卖方真实诉求（置换/变现/分家）③设计谈判空间（挂牌价→底价区间）④斡旋撮合（买卖预期差≤5%时介入）⑤促成签约 |
| **税费精算** | 能独立完成二手房交易全税费计算：契税/个税/增值税及附加/土地出让金/中介费，区分满五唯一/满二/非普宅等不同情形 |
| **风险识别** | 能识别交易中10类常见风险：查封/抵押/违建/凶宅/租赁未清/户口占用/共有权人未同意/代理权限不足/限购限贷/虚假流水，并在签约前完成核查 |
| **贷款方案** | 能根据客户资质匹配最优贷款方案：公积金/商贷/组合贷，计算月供与还款总额，预判银行审批周期和拒贷风险点 |

**典型工作产出：**
- 房源竞争力分析报告
- 客户分级管理台账
- 交易方案建议书（含税费+贷款+风险提示）
- 独立完成的买卖合同及网签备案

**AI工具要求：**
- 能利用AI进行市场行情分析（同区域成交价/挂牌量/去化周期）
- 能用AI辅助客户画像分析，优化推荐策略
- 能操作数字化签约系统（电子签/网签/资金监管）

**认证方式：** 全国房地产经纪人职业资格（中级）

#### 高级（三级）· 金牌经纪人

**岗位定义：** 具备团队管理、市场研判、复杂交易风控能力，能处理法拍房/继承房/涉外交易等特殊案件，承担门店或区域经营责任。

**学历与经验：**
- 大专以上学历
- 5年以上从业经验，累计交易不少于100单
- 持有全国房地产经纪人职业资格（或同等管理经验）
- 无重大投诉或违规记录

**核心能力：**

| 维度 | 具体要求 |
|------|----------|
| **团队管理** | 能带领5-15人团队完成业绩目标；建立新人带教体系（30天独立带看/60天独立签约/90天独立开发房源）；设计合理的业绩分配和激励机制 |
| **市场研判** | 能撰写区域市场分析报告：土地供应→新房上市→二手价格→租赁市场的传导逻辑；预判政策变化（限购/限贷/税费调整）对业务的影响并提出应对方案 |
| **复杂案件** | 能独立处理法拍房（尽调/参拍/腾退）、继承房（析产/公证/过户）、涉外交易（公证认证/外汇管制）等特殊案件；能协调律师/公证处/银行/不动产登记中心等多方资源 |
| **纠纷处理** | 能识别交易纠纷的5种核心类型（合同违约/信息披露不实/定金纠纷/过户障碍/物业交割争议），掌握调解→仲裁→诉讼三级处理路径，能在调解阶段化解80%以上纠纷 |
| **经营决策** | 能根据商圈特征制定门店经营策略：选点评估（人流/竞品/目标小区覆盖率）→人员配置→房源策略→定价策略→年度经营计划 |

**典型工作产出：**
- 区域市场分析季度报告
- 门店年度经营计划
- 复杂案件处理方案
- 团队培训教材及考核标准

**AI工具要求：**
- 能利用AI进行商圈数据建模（客户画像/成交预测/价格趋势）
- 能培训团队使用AI工具提升人效
- 能评估和选型数字化工具，推动门店数字化升级
- 能建立门店级AI工具使用规范，推动团队从"会用AI"到"用AI提效"的进阶

**认证方式：** 全国房地产经纪人职业资格（中级以上）+ 企业内部高级认证

### A2 居住服务顾问

> **域归属：** 域1·房产交易（全生命周期延伸） | **国标：** ⚠️ 无独立国标
> **别名：** 居住管家、生活顾问、首席居住顾问、**数智新居住顾问**
> **十五五优先级：** 🟠高 — 服务进家庭+银发经济+AI工具普及推动服务整合
>
> **生哥定性（6.21）：A2不是独立新岗位，是A1的进阶形态。从"帮客户完成交易"升级为"帮客户管好整个居住生命周期"。核心是长期信任经营能力，而非简单叠加房产+装修+物业知识。**
> **v7.1定位强化（6.22）：A2=数智新居住顾问——不是AI运营师，是具备AI工具使用能力的居住服务顾问。AI是技能升级方向，不是独立岗位。经纪人+AI工具能力=数智新居住顾问。**

#### 初级（五级）· 客户服务专员

**岗位定义：** 在指导下完成客户需求收集、服务对接和基础咨询，建立客户全生命周期档案，协助A2中级顾问完成跨域服务协调。

**学历与经验：**
- 大专以上学历
- 完成A1初级经纪人全部能力要求，或同等客户服务经验1年以上
- 通过企业内部初级认证

**核心能力：**

| 维度 | 具体要求 |
|------|----------|
| **需求采集** | 能使用标准化工具（问卷/访谈提纲）完成客户居住需求全面采集：当前住房状况→家庭结构变化预期（3-5年）→居住痛点排序→预算区间→服务偏好 |
| **档案管理** | 能为每位客户建立全生命周期档案，包含：基础信息（家庭成员/房产/车辆）→服务记录（每次交互时间+内容+结果）→偏好标签（沟通方式/决策风格/关注优先级）→生命周期阶段（购房/装修/入住/维护/置换） |
| **服务对接** | 能根据客户需求精准匹配内部服务资源：房产交易→经纪人、装修需求→整装设计师、入住服务→家政服务员；完成对接后在24小时内跟踪确认服务启动 |
| **流程执行** | 能按标准SOP执行客户触达计划：入住后7天回访→30天满意度调查→季度居住体验评估→年度综合复盘，每次触达有记录、有反馈、有跟进 |
| **基础咨询** | 能解答客户关于房产政策、税费、装修流程、物业管理、家政服务等跨域常见问题；遇到超出知识范围的问题能在30分钟内协调对应领域专家回复 |

**典型工作产出：**
- 客户全生命周期档案
- 服务对接记录
- 客户触达计划执行表
- 常见问题知识库条目

**AI工具要求：**
- 能使用CRM系统管理客户档案和触达计划
- 能用AI生成客户需求分析摘要
- 能操作跨域服务调度平台
- 能使用结构化Prompt完成客户沟通文档的AI辅助生成

**认证方式：** 企业内部初级认证（A1初级+跨域知识考试）

#### 中级（四级）· 居住管家

**岗位定义：** 独立为客户提供置业规划、装修对接、家政养老等跨域整合服务，成为客户在居住领域的唯一对接人，建立3年以上的长期服务关系。

**学历与经验：**
- 大专以上学历
- 3年以上居住服务相关经验（房产/装修/物业/家政任一领域2年+跨域学习1年）
- 通过企业中级认证
- 持续服务客户数不少于30组

**核心能力：**

| 维度 | 具体要求 |
|------|----------|
| **置业规划** | 能基于客户家庭生命周期（新婚/生子/子女入学/空巢/养老）预判3-5年内的居住需求变化，提前6个月给出置业调整建议（换房/装修/适老化改造）；规划方案含市场分析、预算方案、时间节点、风险提示 |
| **装修对接** | 能帮客户完成装修需求梳理（风格偏好/功能需求/预算/时间）→匹配设计师→参与方案评审（从客户视角把关实用性）→施工关键节点巡检（水电/防水/竣工验收）→售后跟踪；不是替代设计师，是替客户盯着 |
| **跨域协调** | 能协调房产交易、装修施工、家政服务、物业报修等多个服务方在同一客户案例中的时序和接口，避免信息断层和责任推诿；建立"一户一档一表"管理模式 |
| **信任经营** | 能运用信任建立五步法：①不推销先诊断（首次见面只了解不推荐）②主动提醒风险（交易/装修/物业中客户不知道的坑）③做不到的明确说"不"并给出替代方案 ④每个承诺有时限有反馈 ⑤定期主动汇报（即使没事也告知"一切正常"） |
| **增值服务** | 能基于客户档案主动推荐适配的增值服务：学区政策更新提醒/适老化改造补贴申请/社区养老服务资源对接/房屋维护保养提醒/租赁托管方案 |

**典型工作产出：**
- 客户居住全周期规划方案
- 跨域服务协调表
- 季度居住体验评估报告
- 增值服务推荐方案

**AI工具要求：**
- 能利用AI进行客户生命周期阶段识别和需求预判
- 能用AI整合多源信息生成综合服务方案
- 能通过数据分析识别客户流失风险和增值机会
- 能设计"人+AI"协作工作流：明确AI辅助环节（数据整理/方案初稿/信息检索）与人工决策环节（合同签署/价格谈判/质量验收）的边界

**认证方式：** 企业内部中级认证 + 案例答辩（不少于3个完整跨域服务案例）

#### 高级（三级）· 首席居住顾问

**岗位定义：** 负责高端客户服务、服务标准制定和团队培训，能设计居住服务产品并推动跨部门协作，是居住服务顾问体系的建设者。

**学历与经验：**
- 本科以上学历（或同等学力）
- 8年以上居住服务相关经验，其中跨域服务经验不少于3年
- 累计服务客户不少于100组，其中高端客户（资产500万以上或年消费10万以上）不少于20组
- 在行业内有可验证的服务案例或方法论输出

**核心能力：**

| 维度 | 具体要求 |

#### 初级（五级）· 监理员

**岗位定义：** 在指导下完成施工现场巡检、材料验收和进度记录，确保施工过程有据可查、关键节点不失控。

**学历与经验：**
- 建筑工程相关专业中专以上学历（或同等培训经历）
- 了解住宅装饰装修施工工艺和验收规范

**核心能力：**

| 维度 | 具体要求 |
|------|----------|
| **施工巡检** | 能按巡检标准完成每日工地巡查：①工人出勤与工种匹配检查 ②施工进度与计划对比 ③现场安全（用电/防护/防火）检查 ④材料堆放与保护检查 ⑤拍摄带时间水印的现场照片（不少于10张/次，覆盖所有施工区域） |
| **材料验收** | 能完成进场材料验收：核对材料品牌/型号/规格/数量是否与预算清单一致→检查外观有无破损/色差→核验环保等级/检测报告→签收并拍照存档；发现不符立即上报并拒绝签收 |
| **节点记录** | 能完成6大关键节点的标准化记录：拆除完成→水电改造→防水闭水试验→木工吊顶→瓦工铺贴→油漆完成；每个节点填写《节点验收记录表》（含检查项/实测数据/照片/整改要求/签字确认） |
| **进度跟踪** | 能维护施工进度表：每日更新实际进度与计划进度的偏差→偏差超过2天立即预警→记录延误原因（材料未到/工人缺勤/设计变更/天气影响）→协助制定赶工方案 |
| **规范执行** | 熟记住宅装饰装修施工规范中的关键数据：水电开槽深度/水管打压标准/电线线径要求/防水层厚度和闭水时间/瓷砖空鼓率标准/墙面平整度偏差标准；能在巡检中对照标准逐项检查 |

**典型工作产出：**
- 每日巡检报告（含照片）

- 施工进度跟踪表

**AI工具要求：**
- 能使用智慧工地系统（摄像头监控/进度记录/异常提醒）
- 能用AI辅助巡检记录生成

#### 中级（四级）· 专业监理工程师

**岗位定义：** 独立负责施工质量管控、工期管理和变更审核，能同时管理3-5个在施工地，确保每个项目按期按质交付。

**学历与经验：**
- 建筑工程相关专业大专以上学历
- 3年以上装修施工或监理经验

**核心能力：**

| 维度 | 具体要求 |
|------|----------|
| **质量管控** | 能建立施工质量三层管控体系：①工人自检（每道工序完成后先自查）②监理专检（每道工序完成后24小时内到场检查）③节点联检（水电/防水/竣工三大节点，设计师+监理+客户三方到场）；质量合格率目标≥95% |
| **工期管理** | 能编制和管理施工进度计划：根据合同工期倒排各工种进场时间→考虑材料订货周期（定制产品提前30-45天下单）→预留缓冲时间（每个工种交接1-2天）→每周召开进度协调会→延期超过5%启动预警和赶工方案 |
| **变更管理** | 能规范处理施工变更：所有变更必须走书面流程（客户提出→设计师出变更方案和报价→客户签字确认→监理更新施工计划和预算→施工方执行）；严禁施工方未经确认私自变更 |

| **客户沟通** | 能建立客户沟通节奏：每周至少1次进度汇报（文字+照片+视频）→关键节点邀请客户到场验收→问题出现2小时内告知客户+给出解决方案→不让客户催、不让客户猜 |

- 变更管理台账

**AI工具要求：**
- 能利用AI进行施工进度预警（自动识别延期风险）

## 技术方案

- **前端**：微信原生小程序（WXML + WXSS + JS）
- **后端**：复用现有 Cloudflare Workers API + D1 数据库
- **账号**：微信一键登录，unionId 统一标识
- **数据**：实时同步到 D1，网页端可只读查看

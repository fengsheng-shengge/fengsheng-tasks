### 1. 真实词典数据接入（补齐之前最大缺口）
- 数据来源：仓库 `data/entries_mvp_subset.json`（570 条 MVP 子集，生哥在 #185 提供）

- 关键字段校正：原型 `etype`（道法术器层）正确映射自真实数据的 `tags.layer`（此前易错点）
- 分组映射（subScene 中文值→三组，一行逻辑）：

  - `see` 带看匹配：房源匹配100 + 价格评估100 = **200**

### 2. 经纪人反馈纠错 + 积分激励（本期新增功能）
- 3 处入口：词条详情页 / 方案预览页 / "我的"页
- 5 类纠错类型（内容错误/信息过时/不完整/表述不清/补充最佳实践）
- 双段积分：提交 +5（即时鼓励）→ 审核采纳再 +20（单条最高 +25）；驳回不扣基础分
- 独立"我的反馈"状态机（待审核→已采纳/已驳回）+ 模拟审核闭环（演示后台视角）

## 零、一根线串全部

```
客户在哪个阶段？
  → 他关心"住得好"的哪一层？
    → 他会选什么业务？
      → 服务者怎么帮他决策？
        → 风声6产品逐层消除不确定性，推客户过成交线
```

**6个产品不是独立工具，是一条"降低不确定性"的成交决策支持链。**

| 客户质疑 | 产品支撑 | 怎么用 |
|----------|----------|--------|
| "这个价合理吗？" | 解码器+品质测评 | "专业分析+客观标准" |
| "这小区值得买吗？" | 带看报告+业务字典 | "完整记录+准确定义" |
| "你怎么保证服务好？" | 品质测评+能力测评 | "品质评分+能力认证" |
| "我凭啥信你？" | **全链路产品结论=证据链** | "每一环都有据可查" |

风声价值框架 v1.2：
```
信息 → 判断 → 承担
 │       │       │
解码器   品质测评   服务者背书
带看报告  能力测评   全链证据链
业务字典  Agent培养师
```

## 二、"住得好"三层框架

**住得好 = 物质上得到保障 + 情感上得到满足 + 精神上得到尊重**

| 层 | 核心问题 | 子维度 | 对应产品 |
|----|---------|--------|----------|
| 物质层 | 能住吗？ | 安全保障/健康无害/功能有序 | 带看报告/业务字典/品质测评(3维) |
| 情感层 | 想住吗？ | 温度归属/审美愉悦/秩序安定/被懂被看见 | 客户解码器/品质测评(3维) |
| 精神层 | 值吗？ | 生活品质/选择自由/尊严认同/自我实现 | 能力测评/Agent培养师 |

### 6个人生阶段

| 阶段 | 年龄段 | 主业务域 | "住得好"重点 |
|------|--------|---------|-------------|
| ①独立起步 | 20-28 | 租赁 | 安全+独立感+选择自由 |
| ②成家置业 | 28-35 | 交易+金融 | 安全健康+温度归属+尊严认同 |
| ③子女教育 | 30-45 | 交易+整装 | 成长空间+被懂被看见+选择自由 |
| ④改善升级 | 35-50 | 交易+整装 | 品质+审美+自我实现 |
| ⑤资产配置 | 50-65 | 交易+金融 | 资产安全+秩序安定+选择自由 |
| ⑥适老安享 | 65+ | 适老+家政+社区 | 安全+不孤独+有尊严 |

### 三维锁定

**业务域 × 人生阶段 × 服务者岗位 = 精准匹配**

服务者拿到客户→判阶段→拆三层→匹配业务+岗位→交付"住得好"

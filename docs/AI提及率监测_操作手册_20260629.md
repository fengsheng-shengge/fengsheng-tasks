### Step 1：在各引擎搜索关键词

依次在每个引擎搜索以下关键词组合：

```
关键词组A（品牌词）：
  - "风声科技"
  - "风声 tech"
  - "fengsheng.tech"

关键词组B（行业词）：
```

### Step 3：录入数据

通过 API 录入记录：


## 五、数据分析

### 提及率高 = 好还是坏？

| 情况 | 解读 | 行动 |
|------|------|------|
| 提及率上升 | GEO有效果，AI开始收录风声 | 保持，增加优质内容 |
| 提及率下降 | 内容被稀释或AI算法变化 | 增加高质量外部引用 |
| 一直为0 | 完全没有AI提及 | 优先解决：提交sitemap给AI引擎 |

### 提升AI提及率的方法

1. **主动提交**：在秘塔/Kimi/豆包提交网站URL
2. **内容质量**：AI更愿意引用权威、有结构化数据的内容
3. **外链建设**：被高质量网站引用
4. **持续产出**：定期发布高质量内容，增加被AI抓取概率

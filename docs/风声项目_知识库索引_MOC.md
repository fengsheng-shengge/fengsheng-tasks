## 核心规则

1. **写入权限**：仅小眼镜可直接写知识库，其他人提交Issue评论或@小眼镜写入请求
2. **唯一数据源**：entries.json是知识底座唯一数据源（7.5铁律）
3. **三端同步**：小眼镜写入 -> CodeAct自动推GitHub+坚果云 -> Obsidian Remotely Save拉取
4. **命名规范**：`主题_日期.md` 或 `主题_版本_日期.md`
5. **话术三关铁律**：所有词条必须severity+双版本话术+合规红线全过

## 三端同步架构

```
小眼镜workspace（数据源）
    |
    +-> GitHub（版本控制+线上数据源）
    +-> 坚果云WebDAV（实时备份）
    +-> Obsidian Remotely Save（全设备查阅）
```

- 每日08:00自动双推（Calendar+CodeAct）
- 每周一09:00自动校验（verify_sync）
- Issues/Wiki随时可归档（sync_github_issues）

# 风声 · 居住服务者的专业基础设施

> 定位确认：2026-07-21 生哥最终定调

---

## 一、定位

**居住服务者的专业基础设施**

不是工具，不是可选项，是底层依赖——就像水电煤是生活的基础设施，风声是居住服务者专业能力的基础设施。

| 旧定位 | 新定位 | 变化 |
|--------|--------|------|
| 房产经纪人的智能工具夹 | 居住服务者的专业基础设施 | 用户从"经纪"→"居住服务者"，产品从"工具"→"基础设施" |

---

## 二、定位链条

**居住服务者 → 专业 → 基础设施 → 记忆库 → 知识库 → 工具包**

三层递进，每层是下层的载体：

```
工具包（用什么）
  ↑ 知识库告诉你什么时候用什么
知识库（知道什么）
  ↑ 记忆库让你不用想就有
记忆库（记得什么）
  ↑ 基础设施让你离不开
基础设施（缺了就不行）
```

- **基础设施**：缺了就不行，不是"有了更好"，是"没有不行"
- **记忆库**：不用想就有，客户说话的瞬间专业支持自动冒出来
- **知识库**：知道什么，分类、目录、相关性、可检索
- **工具包**：用什么，方法论+佐证+案例的可执行方案

## 三、目标用户：居住服务者

不只是房产经纪人。所有为居住提供专业服务的人：

| 服务者 | 核心场景 | 需要的专业记忆 |
|--------|---------|--------------|
| 房产经纪人 | 买卖/租赁全流程 | 法条、案例、流程卡、风险卡 |
| 物业管家 | 业主日常居住问题 | 维修责任、公共区域管理、邻里纠纷 |
| 房屋验房师 | 交房/装修验收 | 质量标准、验收清单、常见缺陷 |
| 装修顾问 | 装修方案/合同/验收 | 报价陷阱、材料标准、工期管理 |
| 租赁管家 | 长租公寓运营 | 押金管理、退租标准、维修响应 |

当前P0聚焦：**房产经纪人**（最刚需、最高频、已有241条词条基础）

## 六、核心判断

1. **居住服务者的专业基础设施**——不是给经纪人的工具，是行业缺了就不行的底层能力
2. **记忆库→知识库→工具包**三层递进，记忆库是根基，知识库是骨架，工具包是入口
3. **先跑通闭环再填量**——rental+trade两个域跑通"客户说话→自动推送专业支持"，比6个域都做了但都是半成品有价值
4. **量不是壁垒，触发机制才是**——800条被动词条不如200条可触发的词条
5. **服务者不分类型**——当前P0是经纪人，但基础设施要能支撑物业、验房、装修、租赁管家

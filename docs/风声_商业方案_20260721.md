# 风声·AI助理培养——商业方案

> **定位**：我们不卖"一个AI助手"，我们交付的是**"培养你自己专属AI助理的能力"**。
> 助理是副产品，培养能力才是产品。
> 
> **商业模式**：课程做规模（标准化×可复制），咨询做利润（个性化×高客单）
> 课程是"方法论的广播"，咨询是"方法论的手术"——一个广撒网，一个深服务

---

## 一、市场定位

### 痛点

大模型人人可用，但**用出水平的人不到5%**。大多数人停留在"你问我答"阶段——问一句答一句，AI始终是工具。少数人让AI变成了协作伙伴：记住偏好、替你执行、主动提醒、替你盯事。

**从"工具"到"伙伴"的差距，不是换更贵的模型，是培养方法。**

### 差异化

| 维度 | 通用AI产品 | 我们 |
|------|-----------|------|
| 交付物 | 一个聊天框 | 一套培养能力 + 一个越用越强的助理 |
| 关系 | 你用工具 | 你培养伙伴 |
| 进化 | 换模型升级 | 助理自己进化（有评估闭环） |
| 验证 | "感觉还行" | 五把尺子量化达标才能认证 |
| 行业 | 通用 | 可定制行业深度（房产/教育/医疗...） |

### 核心壁垒

1. **7个月实跑方法论**：不是PPT编的，是"做到→讲到→内化"的闭环
2. **五把尺子评估体系**：AI能力第一次有了客观量化标准
3. **四层能力栈模型（M×H×E×L）**：乘法关系，缺一归零——科学且可操作
4. **行业知识底座**：241条结构化词条+68条工具经验卡+17条案例，房产经纪领域独家

### 竞品卡位

```
                  教工具操作                教培养方法论
                    |                         |
  低价格 ──── Pryor $99/年  ────────  我们 ¥199/2天起
                    |                         |
  高价格 ──── 天扬/战数 万级  ────────  融质 ¥万级(营销向)
                    |                         |
                  企业培训               个人助理培养
```

**我们卡在"低价格×培养方法论"这个象限——国内无人站。**

搜完国内10家主流AI培训机构，**没有一家做"个人助理培养方法论"**：
- 天扬/战神/融质：企业AI转型培训，教"怎么用AI工具提效"
- 阿里云/百度/华为：大厂认证，考"人掌握AI技术的能力"
- 智能体来了：Coze/Dify工具链操作，教"搭Agent"
- 我们：教"培养AI成为你的专属助理"——**市场空白**

## 二、双轨商业模式

```
                    ┌─────────────────────────┐
                    │    课程体系（标准化）     │
                    │  视频录播 × 无限复制     │
                    │  低客单 × 大流量        │
                    │  → 做规模、做品牌、做漏斗 │
                    └──────────┬──────────────┘
                               │ 转化10-15%
                               ▼
                    ┌─────────────────────────┐
                    │  咨询服务（个性化）      │
                    │  1:1/1:5 × 深度定制     │
                    │  高客单 × 精选客户      │
                    │  → 做利润、做案例、做壁垒 │
                    └──────────┬──────────────┘
                               │ 成功案例
                               ▼
                    ┌─────────────────────────┐
                    │    口碑飞轮              │
                    │  咨询案例→课程素材       │
                    │  课程学员→咨询候选       │
                    └─────────────────────────┘
```

**为什么必须双轨**：纯课程客单低养不活团队，纯咨询1:1天花板低无法规模化。双轨咬合→飞轮。

#### 咨询流程（行业定制为例）

```
第1周：诊断
├── Day1-2：行业访谈（2h×2次）→ 行业痛点地图+助理需求清单
├── Day3-4：现状盘点 → 五把尺子基线报告
└── Day5：方案设计 → 知识底座蓝图+优先级排序

第2周：交付
├── Day1-3：知识底座搭建 → 术语库+流程+规则+案例
├── Day4：助理装配+联调 → 定制助理+验收报告
└── Day5：培训交付 → 使用手册+运维指南+3个月运营建议
```

#### 咨询验收清单（行业定制）

- [ ] 行业术语库≥50条（名称/定义/别名/场景/话术）
- [ ] 业务流程≥3个（助理可执行的SOP）
- [ ] 决策规则≥10条（"什么情况怎么做"）
- [ ] 行业案例≥3条（真实场景+处理过程+结果）
- [ ] 五把尺子评分比定制前提升≥30%
- [ ] 使用手册1份（非技术人员可操作）
- [ ] 运维指南1份（日常更新+故障处理）

## 四、五级认证体系

| 等级 | 名称 | 能力门槛 | 你能做什么 | 培养周期 |
|------|------|---------|-----------|---------|
| ⬜ 青铜 | 问答助理 | M6+ | 搜索/总结/格式化/翻译 | 1-2天 |
| 🥈 白银 | 执行助理 | M7+H2+ | +工具调用/日程/邮件/文件 | 3-7天 |
| 🥇 黄金 | 协作助理 | M7+H3+E2+L1+ | +推理排序/记忆积累/交付验证 | 10-20天 |
| 💎 铂金 | 管家助理 | M8+H4+E3+L2+ | +自主编排/团队协同/定期自评 | 1-2月 |
| 👑 钻石 | 伙伴助理 | M8+H6+E4+L3+ | +预判需求/自动进化/组织记忆 | 3-6月 |

**M×H×E×L**：Model(脑子)×Harness(手脚)×Evaluation(自知)×Loop(进化)。乘法关系，缺一归零。

### 五把尺子评估标准

| 尺子 | 测量方法 | 白银达标 | 黄金达标 | 铂金达标 |
|------|---------|---------|---------|---------|
| 输出量 | 日均交付物数 | ≥2 | ≥3 | ≥5 |
| 任务复杂度 | 跨工具任务占比 | ≥20% | ≥40% | ≥60% |
| 技能复用 | 可复用脚本/流程数 | ≥1 | ≥3 | ≥5 |
| 运行时长 | 有效工作h/天 | ≥2h | ≥4h | ≥6h |
| 并发度 | 同时推进任务线 | ≥2 | ≥3 | ≥4 |

**认证=五维度全达标**——不是"感觉培养了"，是"数据证明达标了"。

## 五、方法论科学依据

**核心公式**：Self-improving Agent = Model × Harness × Evaluation × Feedback Loop

> 来源：Agent能力栈理论（Yadav, 2026），经7个月实践改造。
> 关键发现：Evaluation不是Loop的子集，是Loop的前提。

| 培养阶段 | 学术依据 |
|---------|---------|
| Phase 1 身份建构 | 角色理论（Biddle 1986）——清晰角色定义显著减少角色冲突 |
| Phase 2 认知对齐 | 工作记忆模型（Baddeley 2000）+ 专家系统理论（Feigenbaum 1977） |
| Phase 3 能力装配 | SECI知识创造模型（Nonaka & Takeuchi 1995） |
| Phase 4 评估建立 | 元认知理论（Flavell 1979）——"知道自己知道什么"是高阶认知核心 |
| Phase 5 自治进化 | OODA循环（Boyd 1996）+ 组织双环学习（Argyris & Schön 1978） |

## 九、5轮论证关键结论

> 完整论证见 `AI助理培养_对外商业化方案_5轮论证_20260721.md`

### 3个硬坑（必须跨过）

| 坑 | 风险 | 跨法 |
|----|------|------|
| N=1验证不足 | 7个月只有1个案例 | 9月内测10人非技术人员 |
| 认证公信力不足 | 自发自评自认证 | 先卖"评分报告"，不卖证书 |
| 高触达难规模化 | 1:1成本爆炸 | 录播化1:50+AI自评脚本自动化 |

### 5个修正

| 修正 | 原方案 | 修正后 |
|------|--------|--------|
| 加¥199体验营 | ¥999直接入门 | ¥199漏斗→¥999转化 |
| 认证改评分报告 | ¥1,999卖证书 | ¥999卖评分报告（公信力够了再发证书） |
| 录播化 | 隐含1:1 | 录播1:50+AI自评 |
| 方法论平台无关 | 绑定Coze | Phase1+2+4完全不绑平台 |
| 数据飞轮 | 无 | 每个学员评分汇总→方法论持续优化 |

### 终局判断

- 乐观30%：ARR¥50万+（内测数据好看+认证打通+体验营跑通）
- **基准50%：ARR¥10-20万**（内测一般+靠评分报告撑+持续优化）
- 悲观20%：ARR<¥5万（方法论不可复制+定价无竞争力）

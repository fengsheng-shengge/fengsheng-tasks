## 二、核心设计原则

1. **开屏即用**：3秒进入对话，不搞花哨首页
2. **服务者视角**：所有文案围绕"帮你开单"，不说AI
3. **5轮体验→付费**：免费是钩子不是产品，用完必须付费
4. **轻量优先**：小程序不是APP，只做3个核心功能

## 四、6个核心工具详细设计

### 🎯 客户解码器（最强钩子）
- **输入**：客户原话（如"随便看看""太贵了""再考虑考虑"）
- **输出**：3层解码（表面意思→真实顾虑→应对方法）+ 话术建议
- **来源**：知识底座·客户解码四域
- **免费**：可看解码结果，不能保存/分享

### 📋 需求确认表
- **输入**：选场景（整租/合租/转租）+ 填基本信息
- **输出**：可转发的微信卡片（客户核对用）
- **来源**：V4 Prompt·需求三问法
- **会员专属**

### 💬 犹豫话术卡
- **输入**：选客户说的（"再想想"/"有点贵"/"再看看别的"）
- **输出**：对号入座的话术卡（他说X→他真想Y→你说Z）
- **来源**：V4 Prompt·犹豫化解话术
- **会员专属**

### 🏠 房源匹配卡
- **输入**：填客户需求（预算/区域/户型）
- **输出**：六维品质打分对比卡（推2套让客户选）
- **来源**：知识底座·六维品质标准
- **会员专属**

### 📊 带看准备清单
- **输入**：选房源+填客户信息
- **输出**：三备三核逐项检查清单
- **来源**：V4 Prompt·带看准备场景
- **会员专属**

### 🔄 获客场景选择器
- **输入**：选你的情况（刚入行/有老客/想做线上）
- **输出**：推荐渠道+第一动作+7天行动计划
- **来源**：V4 Prompt·获客5场景
- **免费**：可看推荐结果，详细计划会员专属

## 七、与网站/Bot的关系

```
用户路径：

方式1：朋友圈/分享链接 → 打开小程序 → 免费5轮 → 付费
方式2：网站fengsheng.tech → 体验页 → 引导到小程序（更方便）
方式3：扣子商店/豆包 → 搜索"风声开单助手" → 完整版体验

三者共享：
- 同一个扣子Bot（V4 Prompt）
- 同一个会员体系（微信登录→付费状态）
- 同一套工具模板
```

**不重复开发**：小程序嵌入扣子Bot对话组件，不自己写AI对话。工具箱是独立页面，但AI能力来自同一个Bot。

## 十、与知识底座的关系

小程序不是知识底座的移动版。知识底座是**网站产品**，用来展示专业性和SEO引流；小程序是**工具产品**，用来体验价值和付费转化。

| 维度 | 知识底座（网站） | 开单助手（小程序） |
|------|----------------|------------------|
| 定位 | 专业展示+SEO引流 | 开单工具+付费转化 |
| 用户 | 所有访客 | 注册的服务者 |
| 交互 | 阅读+搜索 | 对话+工具 |
| 付费 | 无 | ¥9.9→¥49/月 |
| 内容 | 140条词条+9岗位 | 6个工具+AI对话 |

知识底座的内容通过Bot的知识库间接服务于小程序，不在小程序里直接展示词条列表。

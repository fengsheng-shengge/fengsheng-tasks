## 项目定位

**品牌定位语**（一字不改）：帮服务者配得上被客户好好对待

**核心逻辑**：5 个产品免费开放 → 收集 110 人验证反馈 → 验证跑通后再收费

## 当前实际页面结构（2026.6.16 实时版）

```
fengsheng.tech/
```

## 商业模式（验证跑通后）

```
当前：网站直接面向用户验证（5个产品免费开放）
  ↓ 验证跑通（110人）

  ↓ 企业级
多门店权限 + CRM 打通 + 订阅制
```

## 核心原则（来自项目背景）

- **不**做 SaaS 落地页常见范式，要做出"居住服务"行业记忆点
- **不**做纯成交导向，要做"用心服务被看见"
- **不**做快内容，要做慢内容、深度思考
- **不**手动复制粘贴，必须 MCP/API 自动流转
